This repository contains my work on the **Machine Learning Specialization** offered by **deeplearning.ai** and **Stanford University** on Coursera, taught by **Andrew Ng**.

## 📋 About the Specialization

The Machine Learning Specialization is a beginner-to-intermediate 3-course program that teaches the fundamentals of modern machine learning and how to apply them to build real-world AI applications.

**What I am learning:**
- Supervised learning (regression, classification, neural networks)
- Advanced learning algorithms and best practices
- Unsupervised learning (clustering, dimensionality reduction, recommender systems)
- Practical techniques used in Silicon Valley for model evaluation, tuning, and data-centric AI

## 📁 Repository Structure

- **Course 1: Supervised Machine Learning - Regression and Classification**
- **Course 2: Advanced Learning Algorithms**
- **Course 3: Unsupervised Learning, Recommenders, Reinforcement Learning**

Inside each course folder you will find:
- Official programming assignments (labs)
- **Extra notebooks** with additional implementations, experiments, and visualizations
- Mathematical explanations and connections to the **Mathematics for Machine Learning** specialization (Imperial College London)

## 🎯 Mathematical Foundations

I am strengthening the theoretical base in parallel by completing:
